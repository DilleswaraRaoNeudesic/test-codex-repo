# Test Repository for Codex Remediation Agent

This repository is a simple test bed used to validate Codex remediation agent workflows: planning, implementing changes, committing, and pushing to a remote.

## Contents

- `one_plus_one.py`: Minimal Python script that computes `1 + 1` and prints the result.

## Usage

Run the script with Python:

```
python one_plus_one.py
```

Expected output:

```
2
```
